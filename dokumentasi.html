Cara Menggunakan Kalkulator

1. Masukkan Fungsi Integrasi (f(x)) ke dalam kolom "Enter integrand f(x)".

2. Masukkan Rentang Integrasi pada kolom "Enter range".

3. Tentukan Ukuran Langkah pada kolom "Enter step".

4. Tekan tombol "Calculate" untuk memulai perhitungan.

5. Lihat hasil perhitungan di bagian "Results".

Rectangular & Trapezoidal Rule

Rectangular Rule dan Trapezoidal Rule adalah dua metode numerik yang digunakan untuk menghitung pendekatan integral dari suatu fungsi matematika di antara dua titik tertentu.

Metode ini membagi daerah di bawah kurva fungsi menjadi beberapa trapesium atau persegi panjang dan kemudian menjumlahkan luas masing-masing trapesium atau persegi panjang tersebut. Pendekatan ini cocok untuk fungsi yang kurva atau tidak terlalu kompleks.

Rumus untuk menghitung integral menggunakan metode rectangular rule adalah sebagai berikut:

\[ \int_{a}^{b} f(x) \,dx \approx \sum_{i=1}^{n} f(x_i) \cdot \Delta x \]

Dibawah ini juga adalah snippet code untuk menghitung integral menggunakan metode rectangular rule: 


const integrateRct = (fn, step, start, end) => {
    // Menghapus data chart yang ada
    removeChartData();
                                        
    let sumRct = 0;
    for (let i = start; i <= end; i += step) {
        // Menghitung nilai fungsi pada titik tertentu (i)
        // Memperbarui data chart untuk visualisasi
        chartData.push({
            x: i,
            y: roundResult(fn(i))
        });
                                        
        // Menambahkan hasil perkalian nilai fungsi dan langkah ke dalam sumRct
        if (i <= end - step) {
            sumRct = math.add(sumRct, math.multiply(fn(i), step));
        }
                                        
        // Menambahkan nilai x ke dalam array label untuk chart
        myLabels.push(i.toFixed(step.toString().replace('.', '').length - 1));
    }
                                        
    // Menghapus data chart jika hasil sumRct mengandung angka kompleks
    if (typeof sumRct !== 'number') {
        removeChartData();
        hideCanvas();
        chartMsg.classList.remove('is-hidden');
    } else {
        chartMsg.classList.add('is-hidden');
    }
                                        
    return `Metode Rectangular: ${roundResult(sumRct)}`
}

Metode ini juga membagi daerah di bawah kurva fungsi menjadi beberapa trapesium, tetapi menggunakan trapesium untuk menghubungkan dua titik sejajar pada kurva. Ini memberikan pendekatan yang lebih baik untuk fungsi yang kurva dan kompleksitasnya lebih tinggi.

Rumus untuk menghitung integral menggunakan metode trapezoidal rule adalah sebagai berikut:

\[ \int_{a}^{b} f(x) \,dx \approx \frac{\Delta x}{2} \left( f(a) + 2 \sum_{i=1}^{n-1} f(x_i) + f(b) \right) \]

Berikut adalah bagian dari kode JavaScript yang mengimplementasikan trapezoidal rule: 


const integrateTrp = (fn, step, start, end) => {
    let sumTrp = 0;
    for (let i = start; i <= end; i += step) {
        // Menghitung nilai fungsi pada titik tertentu (i) dan pada titik selanjutnya (i + step)
        const fnI = fn(i);
        const fnIPlusStep = fn(i + step);

        // Menambahkan hasil trapezoidal rule ke dalam sumTrp
        sumTrp = math.add(sumTrp, math.multiply(math.add(fnI, fnIPlusStep), step, 0.5));
    }

    return `Metode Trapezoidal: ${roundResult(sumTrp)}`
}
Penggunaan Kalkulator

Input dan Batasan

1. Input Fungsi (Integrand)

Input ini mendefinisikan fungsi matematika yang ingin diintegrasikan.
  • Gunakan operator matematika standar: +, -, *, /.
  • Gunakan fungsi matematika dasar: sin, cos, tan, exp, log, dll.
  • Gunakan notasi aljabar: x^2 untuk kuadrat dari x.

2. Input Range (Rentang Integrasi):

Rentang integrasi menentukan interval nilai x di mana integrasi dilakukan.
  • Masukkan nilai numerik sebagai batas bawah (from) dan batas atas (to)
  • Pastikan nilai batas atas lebih besar dari nilai batas bawah.

3. Input Step (Langkah):

Langkah ini menentukan seberapa kecil interval x yang digunakan selama proses integrasi.
  • Pastikan langkah (step) positif.
  • Jangan memilih langkah yang terlalu besar, yang dapat menyebabkan kehilangan detail dalam hasil.
  • Jika angka yang dipilih semakin kecil, maka komputasi yang diperlukan untuk mengahasilkan graphnya juga akan semakin lama
  • Dapat membuat website crash!

4. Output Grafik:

Hasil integrasi akan ditampilkan dalam bentuk grafik.
  • Hasil integrasi menggunakan metode rectangular rule dan trapezoidal rule akan ditampilkan.
  • Jika hasil terdapat angka kompleks, grafik tidak akan ditampilkan, dan pesan kesalahan akan muncul.
Penggunaan Kalkulator

Contoh Penggunaan

Misalkan ingin menghitung Integral tentu berikut:
  • \( \int_{0}^{2} x^2 + 2 \cdot \sin(x) \,dx \)
Dapat di inputkan kedalam kalkulator sebagai berikut:

1. Input Fungsi (Integrand)

\(f(x) = x^2 + 2 \cdot \sin(x)\)
  • Dapat di inputkan kedalam kalulator dengan memasukan:
  • x^2+2*sin(x)

2. Input Range (Rentang Integrasi):

  • from: 0
  • to: 2

3. Input Step (Langkah):

  • Enter Step : 0.1

4. Memahami Hasil

Hasil integrasi akan ditampilkan dalam bentuk grafik dan juga bentuk nilai angka.

Untuk contoh akan di dapatkan nilai:

  • Rectangular Rule: 4.716
  • Trapezoidal Rule: 4.992

Dapat dilihat bahwa nilai yang dihasilkan oleh metode trapezoidal rule lebih mendekati nilai eksak dari integral tersebut.
